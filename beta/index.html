Groov operating environment

Groov Operating Environment connects the end user to blockchain networks using a simple and consistent human-machine UX environment. The UX includes both graphical and conversational experiences powered by AI. The environment offers decentralized user services like identity, wallet, collaboration, storage, and app store. Developers can quickly write and publish apps for various blockchain platforms using Groov ONE SDK interface.
